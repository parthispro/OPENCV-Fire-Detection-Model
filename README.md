# OPENCV-Fire-Detection-Model
Developed an automated fire detection system utilizing Python and OpenCV to enhance real-time surveillance and safety. This project was a collaborative effort, co-designed and implemented alongside a senior team member.

Technical Approach
Computer Vision Pipeline: Utilized OpenCV to process real-time video frames, applying color-space analysis and contour detection to isolate fire-like patterns.

Algorithm Optimization: Developed detection logic to distinguish between static fire-like objects and dynamic fire signatures, reducing false-positive triggers.

Frontend UI: Designed and implemented a responsive user interface using HTML and CSS to display real-time status, detection logs, and automated alert notifications.
Deployment: Integrated the backend detection logic with the frontend interface to provide a seamless monitoring experience.

Achievement & Impact
Presentation: Successfully presented the project at the inter-school technical competition hosted by Khaitan Public School in 2025.
Recognition: Secured 1st Position among competitive entries, evaluated based on technical complexity, detection accuracy, and practical application.
